# Proyecto de Rediseño de Arrocitas

Este directorio contiene el proyecto para el rediseño premium de la página web de **Arrocitas** (https://arrocitas.top/).

## Estructura del Proyecto

Una vez comience la ejecución, aquí encontrarás:
- `index.html`: Página principal rediseñada.
- `css/style.css`: Estilos personalizados premium.
- `js/main.js`: Funcionalidad interactiva.
- `assets/images/`: Directorio para que coloques tus imágenes personalizadas.

## Cómo Colocar tus Imágenes
Cuando iniciemos la construcción, crearemos la carpeta `assets/images/`. Podrás copiar tus imágenes directamente a esa carpeta en este directorio.
